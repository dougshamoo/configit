## Usage

> This small interactive prompt helps users set up gitconfig user info and defaut aliases quickly and easily.

```sh
npm i -g gitconfig-prompt && gitconfig-prompt
```

## Requirements

- Inquirer ^0.11.0
- shelljs ^0.5.3
- validator ^4.4.0
